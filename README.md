# Kasa

Projet 12 : Application de location d'appartements développée avec React.

## Installation

```bash
npm install
npm run dev
```

Ouvrir `http://localhost:5173`

## Technologies

- React + Vite
- React Router
- Sass

## Structure

- `src/components` - Composants UI
- `src/pages` - Pages de l'application
- `src/hooks` - Logique métier
- `src/context` - État global

## Fonctionnalités

- Liste des logements
- Page de détail avec carrousel
- Page À propos
- Design responsive
